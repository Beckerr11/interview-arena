# Interview Arena

Simulador de entrevista tecnica com treino orientado.

## Problema
Candidatos praticam pouco sob pressao e sem avaliacao objetiva.

## Publico
Devs junior/pleno e profissionais em transicao.

## MVP
- Desafios por stack e nivel de senioridade.
- Modo simulacao com timer e contexto de entrevista.
- Rubrica tecnica para autoavaliacao estruturada.
- Historico de tentativas e progresso.

## Stack
- React + TypeScript + Vite
- Node + Express + TypeScript
- CI com lint/test/build/audit

## Setup
1. npm install --include=dev
2. npm run bootstrap
3. npm run dev

## Monetizacao
Plano premium com trilhas e simulados avancados.
